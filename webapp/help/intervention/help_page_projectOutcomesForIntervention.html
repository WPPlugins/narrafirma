Answer questions about project outcomes

The purpose of this NarraFirma page is to think about how the main part of your project (collection, possibly catalysis, and sensemaking) turned out for each participant group.

There are two reasons to answer these questions:

  • You might want to think about how the project went so you can learn from it and plan future projects.
  • Answering questions about project outcomes feeds the recommendations you can see in the Design intervention plans page.
What to do here

For each participant group involved in your project, create a new set of outcomes. Answer the questions about each group.
